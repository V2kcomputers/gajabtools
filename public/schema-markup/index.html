What is Schema Markup? Types, Examples & Complete SEO Guide (2026)

Schema Markup is a type of structured data that helps search engines like Google understand your website content better. By using schema, your site can appear in rich results, which improves your visibility, click-through rate (CTR), and overall SEO performance.

FeatureDetails
TypeStructured Data
FormatJSON-LD
PurposeSEO Optimization
Main BenefitRich Snippets & Higher CTR

Types of Schema Markup

There are many types of schema markup, but the most important ones for SEO are:

  • Article Schema
  • FAQ Schema
  • Breadcrumb Schema
  • Organization Schema
  • WebSite Schema
  • HowTo Schema

1. Article Schema

This schema is used for blog posts and articles.

<script type="application/ld+json">
{
 "@context": "https://schema.org",
 "@type": "Article",
 "headline": "Schema Guide 2026",
 "author": {
   "@type": "Person",
   "name": "Yojna Portal"
 },
 "datePublished": "2026-04-08",
 "dateModified": "2026-04-08"
}
</script>

2. FAQ Schema

This schema is used for frequently asked questions.

<script type="application/ld+json">
{
 "@context": "https://schema.org",
 "@type": "FAQPage",
 "mainEntity": [
  {
   "@type": "Question",
   "name": "What is schema markup?",
   "acceptedAnswer": {
     "@type": "Answer",
     "text": "Schema markup is structured data that helps search engines understand your content."
   }
  }
 ]
}
</script>

3. Breadcrumb Schema

This schema helps display your site navigation in search results.

<script type="application/ld+json">
{
 "@context": "https://schema.org",
 "@type": "BreadcrumbList",
 "itemListElement": [
  {
   "@type": "ListItem",
   "position": 1,
   "name": "Home",
   "item": "https://example.com"
  },
  {
   "@type": "ListItem",
   "position": 2,
   "name": "Schema Guide",
   "item": "https://example.com/schema"
  }
 ]
}
</script>

4. Organization Schema

This schema defines your brand or website identity.

<script type="application/ld+json">
{
 "@context": "https://schema.org",
 "@type": "Organization",
 "name": "Yojna Portal",
 "url": "https://yojnaportal.com",
 "logo": "https://yojnaportal.com/logo.png"
}
</script>

5. WebSite Schema

This schema enables a search box in Google results.

<script type="application/ld+json">
{
 "@context": "https://schema.org",
 "@type": "WebSite",
 "url": "https://yojnaportal.com",
 "potentialAction": {
   "@type": "SearchAction",
   "target": "https://yojnaportal.com/search?q={search_term_string}",
   "query-input": "required name=search_term_string"
 }
}
</script>

6. HowTo Schema

This schema is used for step-by-step guides.

<script type="application/ld+json">
{
 "@context": "https://schema.org",
 "@type": "HowTo",
 "name": "How to Apply Online",
 "step": [
  {
   "@type": "HowToStep",
   "text": "Visit the official website"
  },
  {
   "@type": "HowToStep",
   "text": "Complete registration"
  },
  {
   "@type": "HowToStep",
   "text": "Fill out the application form"
  }
 ]
}
</script>

How to Add Schema Markup to Your Website

  1. Identify your content type (article, FAQ, etc.)
  2. Create schema code in JSON-LD format
  3. Add it inside a <script> tag
  4. Test using Google Rich Results Test

Benefits of Schema Markup

  • Enables rich snippets in search results
  • Improves click-through rate (CTR)
  • Enhances search visibility
  • Helps search engines understand content better

FAQs

What is schema markup?

Schema markup is structured data that helps search engines understand your content.

Is schema a ranking factor?

Schema is not a direct ranking factor, but it improves CTR and visibility.

Which schema types are most important?

Article, FAQ, and Breadcrumb schema are the most important for SEO.
