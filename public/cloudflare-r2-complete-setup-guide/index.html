Cloudflare R2: Complete Guide (What, Uses, Pricing & Setup)

Cloudflare R2 is an object storage service provided by Cloudflare that allows users to store and deliver large amounts of data such as images, videos, backups, and documents efficiently and at a lower cost.

Main Advantage: Cloudflare R2 offers Zero Egress Fees, meaning you do not pay for data transfer when users download your files.

What is Cloudflare R2?

Cloudflare R2 is similar to Amazon S3 and Google Cloud Storage, but it is designed to eliminate high bandwidth costs. It stores data in the form of objects inside containers called buckets.

  • Store files (images, videos, PDFs)
  • Access via public URL or API
  • Integrates with Cloudflare CDN
  • Works well with static websites like Hugo

How Cloudflare R2 Works

  1. Create a bucket
  2. Upload files into the bucket
  3. Each file gets a unique URL
  4. Use that URL in your website or app

Cloudflare CDN automatically caches and delivers files quickly to users worldwide.

Uses of Cloudflare R2

1. Website Media Hosting

Store images, banners, PDFs, and other static files for fast delivery.

2. Backup Storage

Save database backups and website backups securely.

3. Video Hosting

Host small video files for websites and applications.

4. Static Website Optimization

Reduce GitHub or server load by storing assets in R2.

5. App Development

Use R2 as a backend storage system with APIs.

Cloudflare R2 Pricing

Feature Details
Free Storage 10 GB/month
Paid Storage ~$0.015 per GB/month
Data Transfer (Egress) FREE
Requests Free limit + low cost after
Compared to Amazon S3, where data transfer costs can be high, Cloudflare R2 saves significant money due to zero egress charges.

Advantages of Cloudflare R2

  • Zero egress fees
  • Cost-effective storage
  • Fast delivery via Cloudflare CDN
  • S3-compatible API
  • Ideal for static sites (Hugo, Blogger, etc.)

Limitations of Cloudflare R2

  • Not a full video streaming platform
  • Requires basic technical knowledge
  • No advanced media processing

Step-by-Step Setup Guide

Step 1: Create Cloudflare Account

Sign up or log in to your Cloudflare account.

Step 2: Enable R2

Go to Dashboard → R2 → Click "Get Started"

Step 3: Create Bucket

Click on "Create Bucket" and give a unique name (example: my-images)

Step 4: Upload Files

Upload files manually or using API tools.

Step 5: Enable Public Access

Make bucket public or connect a custom domain.

Step 6: Use in Website

<img src="https://your-bucket.r2.dev/image.jpg">

Best Use Case for Developers

If you are running a static website (like Hugo), Cloudflare R2 is perfect for storing images and files separately. This improves website speed and reduces hosting load.

Pro Tips

  • Use custom CDN domain (cdn.yoursite.com)
  • Compress images before upload
  • Use WebP format for better performance
  • Combine with Cloudflare caching
